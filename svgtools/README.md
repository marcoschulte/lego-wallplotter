# prepare svg

## Inkscape

Install plugin https://github.com/Klowner/inkscape-applytransforms

* Open SVG
* Select all
* Path -> Object to path
* Extension -> Modify Paths -> Apply transform

## Illustrator

* Select object
* Transform -> x: 0, y: 0
* "Export as" + use artboards -> svg
